#Documentación de la API

Tengo las dependencias

- express para hacer el servidor HTTP, definir rutas, manejar peticiones y respuestas y añadir middlewares.
- cors que es un middleware para express que controla que dominios externos pueden hacer peticiones al servidor.
- dotenv carga variables de entorno desde un archivo .env a process.env. Sirve para guardar datos sensibles o configurables fuera del codigo como claves API, contraseñas de bases de datos o el puerto del servidor.
- nodemon para que se recargue el servidor automaticamente al guardar cambios.
